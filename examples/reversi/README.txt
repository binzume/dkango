
# ReversiFs

This is a  very useful software to play Reversi game on your Windows Explorer.

## Ussage

- If drive `O` is already in use, give up using reversifs
- Start reversifs command
- Open `O:\` drive in Windows Explorer
- Change the Explorer settings and window size so that the icons are aligned 8x8.
- Also sort folder by name.
- Now you can play reversi on the Explorer!
